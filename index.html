For designating text sizes, instead of using the common “size scale” with animal names as variables for the text size in order of the animals’ own size
Instead have text sizes by animal name, but ordered alphabetically instead
Start with maybe “cat”, “mouse”, “spider”, for small, medium, large text
Then if I need a new one between small and medium, maybe put “horse” in there, halfway alphabetically between “cat” and “mouse”
Takes a moment to think about initially, but if you’ve got a mole, a mouse, and a muskrat, it’s a heck of a lot easier to know their alphabetical order than their size order

:root {
--cat: 12px;
--mouse: 18px;
--spider: 24px;
}

How to make those horizontal rules with diagonal ends

hr {
border: none;
width: 45%;
height: var(--line-height);
margin: 2% 0 0 0;
background:
linear-gradient(to top left, var(--white) 50%, transparent 50%)
100% 0/var(--line-height) var(--line-height),
linear-gradient(to top right, var(--white) 50%,transparent 50%)
0 0/var(--line-height) var(--line-height),
var(--orange);
background-repeat: no-repeat;
}

Reverse it by changing the top left and top right properties to bottom left and bottom right

More flexible version to above: use cli-path to form a complex polygon. Very useable site at https://bennettfeely.com/clippy/ or use the following to form the clipped horizontal rules

clip-path: polygon(5% 0, 95% 0, 100% 100%, 0% 100%);

array.sort((a,b) => a-b);
